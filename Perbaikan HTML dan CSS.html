Berikut adalah versi sudah dikoreksi dan dirapikan dari kode HTML & CSS kamu:

html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Form Input</title>
  <style>
    form {
      background: blue;
      color: white;
      padding: 20px;
      width: 300px;
      margin: 50px auto;
      border-radius: 10px;
      font-family: Arial, sans-serif;
    }

    input[type="text"], input[type="submit"] {
      display: block;
      width: 100%;
      padding: 10px;
      margin-top: 10px;
      border: none;
      border-radius: 5px;
    }

    input[type="submit"] {
      background-color: white;
      color: blue;
      font-weight: bold;
      cursor: pointer;
    }

    input[type="submit"]:hover {
      background-color: lightgray;
    }
  </style>
</head>
<body>
  <form>
    <label for="nama">Nama:</label>
    <input type="text" name="nama" id="nama" placeholder="Masukkan nama">
    <input type="submit" value="Kirim">
  </form>
</body>
</html>

Perbaikan yang dilakukan:

  • <!DOCTYPE <html> diperbaiki jadi <!DOCTYPE html>

  • <styyle> diperbaiki jadi <style>

  • Penambahan meta charset dan title di <head>

  • <input> tidak membutuhkan tag penutup, dan atribut type dan value harus menggunakan =

  • Ditambahkan styling agar tampilan lebih rapi dan responsif

  • Label ditambahkan untuk aksesibilitas dan keterbacaan

Oke Imay! Berikut langkah-langkah yang bisa kamu ikuti untuk membuat dan mengunggah file ke GitHub secara lengkap:

✅ Langkah 1: Buat Repositori di GitHub

  1. Buka https://github.com

  2. Klik tombol + di kanan atas → pilih New repository

  3. Isi nama repositori, misalnya: form-koreksi-html-css

  4. (Opsional) Tambahkan deskripsi seperti: "Perbaikan sintaks HTML & CSS sederhana dari tugas."

  5. Pilih: Public → centang Add a README file

  6. Klik Create repository

✅ Langkah 2: Upload File HTML ke Repositori

  1. Setelah repositori jadi, klik tombol Add file → Upload files

  2. Siapkan file .html di komputermu (misalnya bernama form.html), lalu drag & drop file ke halaman GitHub

  3. Scroll ke bawah, beri pesan commit: Upload file HTML hasil koreksi

  4. Klik Commit changes

✅ Langkah 3: Salin Link File

Setelah berhasil upload, klik nama file form.html, lalu salin URL dari address bar.
